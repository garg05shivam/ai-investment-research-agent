# AlphaLens 🚀

AI-Powered Investment Research Agent built using React, Node.js, Gemini, and LangGraph.

## Features

* Company Research Analysis
* Financial Assessment
* Risk Evaluation
* Investment Recommendation
* Multi-Agent Workflow
* LangGraph Orchestration
* Gemini AI Integration

## Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS

### Backend

* Node.js
* Express.js

### AI

* Gemini API
* LangGraph

## Agent Workflow

Research Agent
↓
Finance Agent
↓
Risk Agent
↓
Decision Agent

## Project Architecture

Frontend (React)
↓
Backend API (Express)
↓
LangGraph Workflow
↓
Gemini AI
↓
Investment Recommendation

## Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
